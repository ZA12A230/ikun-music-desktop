# LX-Music 免安装版 - 使用说明

## 📦 当前状态

此文件夹包含：
- ✅ 完整的源代码（含所有新增功能）
- ✅ 编译后的 JavaScript 文件
- ✅ 构建脚本和配置
- ✅ 所有必要资源文件

## ⚠️ 待完成步骤

由于网络限制，Windows 可执行文件（electron.exe）需要在 Windows 系统上下载和构建。

## 🚀 在 Windows 上构建（仅需一次）

### 1. 准备工作
下载此 `build` 文件夹到你的 Windows 电脑

### 2. 安装 Node.js
下载地址：https://nodejs.org/ (推荐 v20 LTS)

### 3. 开始构建
打开 Windows Terminal 或 PowerShell，进入 build 文件夹：

```bash
cd build
.\build-portable.bat
```

### 4. 等待完成
- 首次构建会自动下载 Electron（约 100MB）
- 整个过程可能需要 10-20 分钟
- 请保持网络连接

## ✨ 构建完成后

`build\portable` 文件夹就是你的免安装版！

### 使用方法：
1. 进入 `build\portable` 文件夹
2. 双击 `LX Music.exe` 启动
3. **无需安装，直接使用！**

### 便携特性：
- 所有配置和数据存储在 `portable\LxDatas` 文件夹
- 移动整个 portable 文件夹到其他电脑/磁盘也能正常使用
- 删除 portable 文件夹即可完全卸载

## 🎯 已包含的功能

1. **动态背景** - 根据播放歌曲封面自动生成模糊背景
2. **液态玻璃效果** - 全局毛玻璃和半透明效果
3. **定时关闭** - 播放详情页定时关闭功能
4. **第三方同步服务器** - 支持 https://lxsync.5had0w.com/
5. **网盘备份** - 多种网盘备份支持（需后续对接 API）
6. **开机自启动** - 可在设置中开启
7. **开发者** - G佑
